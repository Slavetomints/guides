Vocabulary

Alias
A shortcut created in the shell to simplify the execution of commands. For example, `alias john='/usr/sbin/john'` allows the user to call John the Ripper more easily on ParrotOS.
bcrypt
A hashing algorithm specifically designed for securely hashing passwords, featuring built-in salting and a configurable work factor to make brute-force attacks more difficult.
Brute Force
A password-cracking method that involves trying every possible combination of characters until the correct password is found. Brute-forcing is computationally intensive but effective for shorter passwords.
Ciphertext
The encrypted version of plaintext. For example, the ciphertext of "Hello World!" in Base64 is "SGVsbG8gV29ybGQh".
Hash
The fixed-size output produced by a hashing algorithm. For example, the SHA-256 hash of "password123" is "ef92b778bafe771e89245b89ecbc8c3cc9a89725e3dbd09b94e8f9f374996645".
hashID
A simple CLI tool to determine the format of an unknown hash.
Hashing
A process that applies a mathematical algorithm to data (like passwords) to transform it into a fixed-size output called a hash. Hashing is a one-way process, meaning it cannot be reversed to retrieve the original data.
John the Ripper
An offline password-cracking tool that supports a wide variety of hash algorithms. Its flexibility and features, enhanced by the `Jumbo John` version, make it one of the most popular tools for password security testing.
Johnny
The graphical user interface (GUI) version of John the Ripper, included in ParrotOS, which provides a user-friendly way to perform password-cracking tasks.
Jumbo John
An extended version of John the Ripper that includes additional features and support for more hashing algorithms and password-cracking techniques.
Kali Linux
A Debian-based Linux distribution designed for cybersecurity tasks such as penetration testing and digital forensics. It comes preinstalled with tools like John the Ripper.
MD5
A widely used cryptographic hashing algorithm that produces a 128-bit hash. It is considered insecure for many uses because it is vulnerable to collision and preimage attacks.
Package Manager
A tool used in Linux and other operating systems to install, update, and manage software packages. Examples include `apt` for Debian-based systems and `yum` for Red Hat-based systems.
ParrotOS
A security-focused Linux distribution, similar to Kali Linux, offering tools for penetration testing, vulnerability assessment, and forensics. It includes a GUI version of John the Ripper called `johnny`.
Password Cracking
The process of recovering plaintext passwords from their hashed versions using methods such as wordlists or brute force.
Plaintext
The unencrypted version of data. For example, the plaintext of the Base64 encoded string "SGVsbG8gV29ybGQh" is "Hello World!".
rockyou.txt
A famous wordlist containing about 14.5 million passwords, derived from a 2009 data breach at `rockyou.com`. It is commonly used in password-cracking tools.
SHA-256
A member of the SHA-2 family of cryptographic hash functions. It produces a 256-bit hash and is commonly used for secure password storage and data integrity checks.
Wordlist
A text file containing a list of potential passwords used in password-cracking attacks. Tools like `John the Ripper` and `Hashcat` use wordlists to compare hashed values to known plaintext values.
